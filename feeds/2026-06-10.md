💬 [@shimst3r](https://famichiki.jp/@shimst3r) 
It's nice to end alt text with a period so there's a pause before a screen reader reads the next thing, the same as any text.

Alt text is read as one continuous string so line breaks don't matter. There are no semantic structures within so there can't be lists.

You can certainly use a free screen reader (VoiceOver, Windows Narrator, NVDA, TalkBack, Orca) to navigate and listen to posts and hear what a post, including alt text, sounds like. Almost any accessibility checking tool will flag images missing alt text and some will call out common alt text mistakes, like file names.
There are also bookmarklets and browser extensions that can temporarily display the alt text of images to make reading them easier.
