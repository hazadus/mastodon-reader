💬 Learning Notes – Go Language – Day 02

constant

 A constant can be declared asconst n = 5000
It can not be changed in runtime.   Variables

 var declares 1 or more variablescan declare multiple variables in same lineif no value given for a variable, its default value is 0var f string = “apple” and f := “apple” are same

package main

import "fmt"

func main() {

    var a = "initial"
    fmt.Println(a)

    var b, c int = 1, 2
    fmt.Println(b, c)

    var d = true
    fmt.Println(d)

    var e int
    fmt.Println(e)

    f := "apple"
    fmt.Println(f)
}
   Inputs

 
 scan(), scanf(), scanln() are three functions used for getting input.
  scan()

 
 scan() – gets input as raw. stores in given variables as space separated values. 

var i int

fmt.Print("Type a number: ")
fmt.Scan(&i)
fmt.Println("Your number is:", i)
 To get more values, use more variables. 

var i,j int

fmt.Print("Type two numbers: ")
fmt.Scan(&i, &j)
fmt.Println("Your numbers are:", i, "and", j)
 give input as “1 2” then it assigns as i = 1 and j = 2 

 &i and &j are memory locations of i and j.

scanf()

 
 fmt.scanf() gets the inputs and stores them based on the determined formats for its arguments. like %d, %s etc. 

package main
import ("fmt")

func main() {
  var i,j int

  fmt.Print("Type two numbers: ")
  fmt.Scanf("%d %d",&i, &j)
  fmt.Println("Your numbers are:", i, "and", j)
}  
   scanln()

 
 same as scanf() but fmt.scanln() stops scanning for inputs at a newline
    If/Else

 
 If/else if/else loop is used for control flow. 

 example: 

package main

import "fmt"

func main() {

    if num := 9; num < 0 {
        fmt.Println(num, "is negative")
    } else if num < 10 {
        fmt.Println(num, "has 1 digit")
    } else {
        fmt.Println(num, "has multiple digits")
    }
}  
   for loop

 
 For loop has classic initial/condition/after for loop. 

for i := 0; i < 3; i++ {
        fmt.Println(i)
    }
